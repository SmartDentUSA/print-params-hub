

## Plano: Inserir chaves da Loja Integrada

Vou solicitar as duas chaves usando a ferramenta de secrets do Lovable. Cada uma abrirá uma caixa de texto segura aqui no chat para você colar o valor.

### Secrets a configurar

| # | Secret | Descrição |
|---|--------|-----------|
| 1 | `LOJA_INTEGRADA_API_KEY` | Chave da API da loja |
| 2 | `LOJA_INTEGRADA_APP_KEY` | Chave da aplicação/integrador |

Nenhuma alteração de código necessária — apenas atualização dos valores dos secrets existentes.
